# Docker crash course

Create a new image:
- `docker build .`

List images:
- `docker images`

Start and enter an image:
- `docker run -it <image-id>`

List containers:
- `docker container ls`

Delete stopped containers:
- `docker container prune`

Clean up all unused docker things:
- `docker system prune -a`
